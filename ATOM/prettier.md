# Prettier
> 代码格式化插件，多端通用  
[https://prettier.io](https://prettier.io)
## 配置
> 支持全局、项目配置以及eslint配置
- 项目
> prettier-atom会自动检索配置文件，可在项目目录下建立文件 .prettierrc，支持json等多种格式，如
```json
{
  "trailingComma": "es5",
  "singleQuote": true,
  "arrowParens": "always",
  "printWidth": 70
}
```
- 全局
> 在 ~ 处新建文件 .prettierrc
- eslint
> 感觉多此一举，回头补充吧
